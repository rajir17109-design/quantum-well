# ⚛️ QuantumWell — AI Productivity & Wellness Assistant

A multi-agent AI app that tracks your diet, workouts, and tasks through natural conversation.

Built for **Google Cloud Gen AI Academy APAC 2026 — Cohort 1 Hackathon** by **Team Quantum Coders** 🚀

## 🛠️ Tech Stack
- Gemini 2.5 Flash + Vertex AI
- FastAPI + Cloud Run
- HTML/CSS/JS Frontend

## 🤖 Agents
- Orchestrator — routes requests
- Diet Agent — tracks meals & calories
- Workout Agent — logs exercises
- Task Agent — manages to-dos
- Progress Agent — daily summary
